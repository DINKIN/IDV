An in-depth look at an ISL that is used to generate Google Earth KMZ products. This uses the [gen::islTag import], [gen::islTag procedure], [gen::islTag call] and [gen::islTag output] tags.
  1. To start make an [ht::filename example] directory.
  2. Right click on the two links below and save the files to the [ht::filename example] directory.
    generate.isl
    example.isl
  3. You will need to create 2 bundles in the [ht::filename example] directory:
    [ht::filename goeseast.jnlp]
    [ht::filename goeseastir.jnlp]
  4. To run this: 
    idv /home/idv/example/example.isl

Here is the list of bundles: [gen::isl example.isl]

Here is the main isl file: [gen::isl generate.isl]
