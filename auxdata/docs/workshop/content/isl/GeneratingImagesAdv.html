More examples on generating image and movie.
  1. First of all save this isl2.xidv file to your isl directory. This is a bundle with two view windows. You can load this bundle into the IDV and see what this bundle doing.
  2. Run this file: [gen::isl image7.isl]
  3. Now, change directory to the home directory (cd) and run: 
    runIDV <%workshop.homedir%>/isl/image7.isl
  4. What happened?
  5. Note, two images were created. If we want to generate a single image which combines these two images, we'll use the combine property: [gen::isl image8.isl]
  6. The IDV gives you an easy way to generate a publication quality combined image.
  7. The next two isl examples will provide some details of more advance isl scripts.
    • How to define an ISL procedure to generate image. [gen::isl procedure.isl]
    • How to use the displayproperties tag to change two end points of a cross section in the bundle RUC.xidv. [gen::isl contourCrossSection.isl]
