Let's do more with bundles.
  1. Lets just save off the data sources loaded.
    1. If you don't have any data and displays load in the [ht::filename workshop.xidv] bundle with the [ht::menu File {Open}] menu.
    2. Let's also load in the Sample AVN model data from the Catalog chooser.
    3. Select the menu [ht::menu File {Save As}].
    4. Enter [ht::filename justdata.xidv] as a file.
    5. Check off [ht::button Views] and [ht::button Displays] and press [ht::button Save].
    6. Open this bundle with the menu [ht::menu File {Open}]. Remove previous data and displays.
  2. Let's just save off the displays.
    1. Create a Color-Shaded Plan View from the [ht::param temperature] field from the AVN Data Source.
    2. Save the bundle with the menu [ht::menu File {Save As}].
    3. Enter [ht::filename justdisplays.xidv] as a file.
    4. Check off [ht::button Views] and [ht::button Data Sources].
    5. Check on [ht::button Displays].
    6. Press [ht::button Save].
    7. Remove the displays with the [ht::menu {Remove all displays}] menu.
    8. Open the [ht::filename justdisplays.xidv] bundle with the menu [ht::menu File {Open}].
    9. !!! Make sure you select "[ht::button Don't remove displays and data]".
    10. When we load in a bundle with only displays the IDV will prompt you for the data.
    11. Choose the [ht::param temperature] field from the RUC data source.
