{Storm Track Chart Control} { In the storm chart tab, it is displaying the charting of a single or multiple parameters. By default, the storm chart tab is the forecast time chart. But user can add a forecast hour chart control (Edit->Add Forecast Hour Chart). To do the charting of the storm track, user need to select one or moe Techs including observation in the tech list, select the forecast time or hour, select the desired parameters, and turn on or off the Use Difference check box.
