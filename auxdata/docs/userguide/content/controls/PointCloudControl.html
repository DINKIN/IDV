{Point Cloud Control} { The Point Cloud control is a volume renderer of a set of vertices in a three-dimensional coordinate system. It shows a large number of points that are typically intended to be the representative of the external surface of an object. The control window contains three tabs. The first shows the point cloud control [ht::cimg images/PointCloudControlGui1.gif] The second shows the clipping control, it is a simplify version of draw control [ht::cimg images/PointCloudControlGui2.gif] The third tab shows the shape information of the clipping [ht::cimg images/PointCloudControlGui3.gif] } {{Color By} { You can select a parameter to be the color by value. }} [ug::colortable] {{Visible Range} { You can set the range of data that is shown by clicking this button and changing the value in the pop up window. }} {{Point Size} { You can change the size of the displayed points with the Point Size widget. Enter a point size and hit return. }} {{Clipping} { You can enable the clipping by click the button and then reload the display to show the point cloud display inside or outside the clipping boundary. The clipping boundary is done in the clipping control. }}
