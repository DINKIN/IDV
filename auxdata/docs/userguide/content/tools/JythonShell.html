[ug::head {Jython Shell}] The Jython Shell is an interactive command line interface to the IDV Jython interpreter.

The User Interface

Library Commands

  • Select data. 
    data = selectData();
    Brings up the field selector. Returns a visad.Data object.

  • Select a data choice. You would want to do this if you want to access its name, etc. 
    dataChoice = selectDataChoice();
    Brings up the field selector. Returns a DataChoice object. You can access the data with: 
    data = dataChoice.getData(None);

  • Create a display. 
    display = createDisplay('some display type', data_or_datachoice);
    This creates a new display with the given data or data choice. The display type can be inserted through the {ht::menu Edit {Insert Display Type}] menu.

    Once you have created the display you can remove if with: 

    display.doRemove();
