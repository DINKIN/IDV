Get the Nth grid in time series fieldimpl, a VisAD FieldImpl; user must be sure input is a suitable data field. returns a FlatField. Nth is an integer, >=0, <= max index of grid time series.

getSliceAtAltitude(fieldimpl, alt, unit):

Extract a 2D horizontal slice from a 3D grid at the given altitude level is a real number; if unit is supplied, it must be compatible with meters (ft, fathoms, etc) param fieldimpl is a VisAD FieldImpl which may have one or more time steps.

getSliceAtLevel(fieldimpl, level):

Extract a 2D horizontal slice from a 3D grid at "Level." level is a real number; must be appropriate for the grid. param fieldimpl is a VisAD FieldImpl which may have one or more time steps.
