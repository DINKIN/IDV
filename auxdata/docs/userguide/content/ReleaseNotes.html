Integrated Data Viewer
Version <%idv.version%>
The items below reflect changes since the 3.1u1 release. A new version of the netCDF-Java library (4.2.34) is included in this release. See the netCDF-Java change log for a complete list of changes. The IDV now can plot 2D and 3D gridded trajectory. The 2D and 3D grid trajectory are derived data choices based on the horizontal velocity compnoents and vertical velocity component. The trajectory is colored by a user selected parameter. There are three options in the Grid Trajectory Control to initialize the trajectory intial area. Added a user interface to change the frequency of the contour labels. The density of the label can now be increased or decreased along the contours or horizontally across the display. Fixed broken webcam links available in the IDV. The IDV Scripting Language (ISL) has new capability to export data available in ObsListControl (e.g. a Point Data List display for METAR data). The IDV Scripting Language (ISL) image tag has a new animation_index attribute to specify multiple images in the animation. There is a new RAMADDA section in IDV training workshop. For a list of outstanding known problems, see the Known Problems page.
