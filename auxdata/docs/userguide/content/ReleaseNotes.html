Integrated Data Viewer
Version <%idv.version%>
The items below reflect changes since the 3.0u2 release. Upon IDV start up, "PermGen" memory allocation is done automatically according to the user's machine specifications. The default PermGen memory allocation can be changed through the User Preferences ( [ug::menu Edit Preferences] menu) under the System tab. A new version of the netCDF-Java library (4.2.34) is included in this release. See the netCDF-Java change log for a complete list of changes. There are two new ways to provide latitude and longitude labels on each view. First, you can place labels along the edges of the box axes through the [ug::menu View Properties] menu under the [ug::button {Horizontal Scale}] tab.
