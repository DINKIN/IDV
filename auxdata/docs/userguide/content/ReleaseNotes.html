The items below reflect changes since the 2.9u3 release. Upon IDV start up, memory allocation is done automatically according to the user's machine specifications, also IDV memory allocation can be changed via Edit --> Preferences, System tab. The IDV can now be installed as a Mac OS X bundle. The IDV add new user interface to add logo easily via Edit --> Preferences, View tab . netCDF-Java change log Jython library 1.5.2 is in the IDV library. New version of ViSAD library is installed. Added user interface to resize the contour labels (with the consideration of the aspect ratio in the IDV), changing font, show and hide, and align (in globe view too). Added Ensemble grids UIs to support selecting all or partial members. Added Ensemble grids diagnostic functions, including Average, Highest, Lowest, Mode, Percentile, Range, and Standard Deviation. Added a new display type to do the Hovmoller display. Added 2D parameter as topography displays, including Color-Shaded Parameter As Topography, Contoured Parameter As Topography, Color-Filled Contoured Parameter As Topography, and Radar Sweep As Topography. Added support for GrADS dataset. Several bug fixes were made in this release, including:
  • Fixed wind barb issue.
  • Fixed hurricane track color control issue.
  • Fixed CAP profile display bug.
For a list of outstanding known problems, see the Known Problems page.
