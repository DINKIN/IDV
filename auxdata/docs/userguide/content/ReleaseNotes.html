Version <%idv.version%>
The items below reflect changes since the 2.4 release. This is mainly a maintenance release with some bug fixes and performance improvements. The next release after <%idv.version%> will involve several changes to the underlying infrastructure:
  • Use of the Java-netCDF 4.0 library. This will provide limited support for HDF-4 and HDF-5 files.
  • Use of Java 6 (JDK 1.6) in the installers and Web Start.
 There is a new chooser under the Radar tab of the Data Chooser window that allows you to access radar volume data (Level II) from a THREDDS Data Server (TDS). For a list of outstanding known problems, see the Known Problems page.
