Integrated Data Viewer
Version <%idv.version%>
The items below reflect changes since the 2.3 release. Users can save properties of display controls in favorite display settings. These can be used when displays are created or shared between controls. See the section on Display Settings in the User's Guide. Users can now set the vertical scale range in the Cross Section and Data Transect Display controls. Users can show the aggregation of point observations through time. This is useful for showing how many observations accumulate through time (e.g. earthquake epicenters, lightning plots) in a given area. For more information, see the Point Data Plot control documentation. The Jython library has been enhanced with many new functions. Most of the GEMPAK grid diagnostics have been implemented. For a complete list, see Jython Library Functions. The IDV can now read in GEMPAK grid files from local disk and from a THREDDS Data Server (TDS). Only grids with no packing, Grib 1 or Grib 2 packing are supported at this point. You can now save remote grids to your local system and/or use the IDV to subset these remote grids. When you load in a gridded dataset, you can use the [ug::menu {Write Grid}] menu from the Data Source properties menu to save the grid. The data will be saved in a netCDF file using the CF 1.0 convention. This same mechanism can be used to subset a local grid and/or convert from the native format to netCDF. Several improvements have been made to the Text Point Data Source. Point data stored in comma-separated value (CSV) text files or Excel spreadsheets can be read into the IDV. The data source has an metadata GUI which can be used to specify the parameters, the units and the structure of the file. For more information, see the Text Point Data File page in the User's Guide. There is a new Field Mask option which allows you to define a mask for a dataset such that only certain fields are shown in the field selector. The field mask can be used for a particular dataset, or can be applied to similiar datasets. Access this feature through the Data Source properties menu using the [ug::menu {Write Field Mask Plugin}] option. A new interactive Jython Shell can be accessed under the [ht::menu Edit Formulas {Jython Shell}] menu. This shell allows you to interactively write new procedures, load in data and create displays. The Jython library GUI has been updated to use a tree view instead of tabs to access the various libraries. It now has a search function and facilities for automatically creating formulas from your Jython methods. For more information, see the Jython Library documentation.
