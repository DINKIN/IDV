The items below reflect changes since the 2.9u2 release. The Java Web Start version of the IDV has updated DigiCert certificates lasting for one year. netCDF-Java change log Added a function to extract longitudes out of a NWP Grid. Improved and cleaned up workshop documentation. Webcam links have been cleaned up. The new VisAD version includes wind barb improvements. According to NASA: The JPL One Earth "server no longer provides full WMS services for any of the datasets. Furthermore, the MODIS daily mosaics are no longer being updated. The tiled WMS access, described in the Tiled WMS and Google Earth KML support will stay operational. This includes access to the archived of MODIS daily mosaics, which were built from 2006 to the end of 2010."

We have updated the IDV to point to the NASA Earth Observations (NEO) WMS This isn't an exact replacement, but it will do.

In situations where users have smoothed a grid, they can now revert back to an unsmoothed grid without having to reload the data. In addition, users can longer smooth grids over topography fields. In this specific situation the user must smooth the grid via the IDV Jython formulas. Several bug fixes were made in this release, including:
  • Probe control fixes for the fly-through.
  • Fixed wind barb issue.
For a list of outstanding known problems, see the Known Problems page.
