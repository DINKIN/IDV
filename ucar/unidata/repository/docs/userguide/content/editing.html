[ht::head {Editing Entries and Metadata}] <%nochildlist%> Once an entry is created you can edit its name, description, time range and lat/lon bounding box with the edit icon in the toolbar: which takes you to the entry edit form: [ht::cimg images/editform.jpg {Entry Edit Form} {border=0 width=600}] Along the top are other edit links that allow you to view, edit and add metadata and define access control to this entry.

If the description of an entry begins with "" then RAMADDA uses the description text as a wiki page when displaying the entry in the default HTML view. See here for more information on wiki text in RAMADDA. Adding Metadata In RAMADDA entries have some basic metadata that can be edited in the edit form, e.g., name, description, lat/lon bounding box, time range, etc. One can also add any number of other metadata elements to an entry.

Clicking on the "Add Metadata" link takes you to the Add Metadata form where you can select the type of metadata to add. e.g., Tag: [ht::cimg images/addmetadata1.gif {Add Metadata Form} {border=0 width=600}] You can then specify the metadata value or values: [ht::cimg images/addmetadata2.gif {Add Tag Metadata Form} {border=0 width=600}] You can upload a file as an attachment to an entry: [ht::cimg images/addthumbnail.gif {Add Thumbnail} {border=0 width=600}] If you create a "thumbnail" attachment that is an image it will be displayed when the entry is listed: [ht::cimg images/thumbnails.gif {Thumbnail Images} {border=0 width=600}] Clicking on a thumbnail image will display the full size image: [ht::cimg images/thumbnailexpanded.gif {Expanded Thumbnail} {border=0 width=600}] [ht::foot]
