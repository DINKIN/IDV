The [ug::chooser Radar] chooser contains three sub-tabs that allow you to choose NEXRAD Level III data on remote ADDE servers or NEXRAD Level II/III data from your file system or a remote THREDDS Data Server (TDS). [ht::cimg images/TDSRadarChooser.gif {TDS Radar Chooser}] The [ug::chooser {NEXRAD - Remote}] chooser allows you to choose Level II, Level III or Terminal Doppler Weather Radar (TDWR) data.
